# MRD Brindes - Sistema PWA

Sistema profissional para gerenciamento de clientes, rotas, prêmios e cobranças de raspinhas.

## Como subir para o seu GitHub

Siga estes passos exatos no seu terminal local:

1. **Abra o terminal** na pasta do projeto.
2. **Execute os comandos**:

```bash
git init
git add .
git commit -m "Final version - Mobile Optimized"
git branch -M main
git remote add origin https://github.com/lirofyu-web/studio.git
git push -u origin main
```

## Destaques

- **Mobile First**: Interface otimizada para uso com uma mão no celular.
- **Offline Ready**: Funciona sem internet (PWA) usando Firebase Persistence.
- **Recibo Térmico**: Impressão direta para impressoras Bluetooth de 80mm.
- **Animações de Sucesso**: Feedback visual em todas as operações de salvamento.
