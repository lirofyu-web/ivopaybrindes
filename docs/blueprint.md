# **App Name**: IVOPAY Gestão

## Core Features:

- Add Client: Add new clients with name, phone (with WhatsApp link), address, city, and current location.
- Dashboard: Dashboard displaying key metrics related to Clients, Prizes, Billings, Reports, Expenses, and Settings.
- Routes Management: Manage delivery routes for the raspinhas, optimizing for efficiency.
- Prizes Management: Create, edit, and track the prizes associated with the raspinhas.
- Billing Management: Manage billing and payment tracking for raspinhas sales.
- Expense Tracking: Track expenses related to the operation of raspinhas sales.
- Report Generation: Generate sales and financial reports to gain insights into the business.

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2) to inspire trust and reflect a clean, modern look.
- Background color: Very light blue (#F0F8FF) to keep things clean and unobtrusive.
- Accent color: Orange (#FF9933) to add visual interest to calls to action and important information.
- Body and headline font: 'PT Sans', a humanist sans-serif offering a balance of modern style and legibility, well-suited for both headings and body text.
- Use clear, simple icons to represent dashboard items and navigation elements.
- Clean and structured layout to present data in an easily digestible format.
- Subtle animations to improve user experience, such as transitions between sections and loading indicators.
